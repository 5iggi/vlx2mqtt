## What is VLX2MQTT?

VLX2MQTT is a LoxBerry plugin and a Python-based MQTT bridge for **VELUX KLF200 / Homecontrol IO**.
It connects directly to the KLF200, reads positions and states from windows and shutters, publishes them via MQTT, and processes MQTT commands for control.

### Highlights

- MQTT bridge for **VELUX KLF200 / Homecontrol IO**
- status and control for windows and shutters
- optional indirect rain detection for supported windows
- recovery / power-cycle integration for unstable KLF connections
- compact operational logging for production environments
